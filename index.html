Syntax Rules

  • Property names, String values must be in double quotes

Basic JSON

  • Syntax is identical to Javascript Object Literal BUT is NOT and object literal.
  • Needs to be converted into a JavaScript Object to be used as an object literal
JSON Example
{
    "firstName": "Alice",
    "lastName": "Wonderland",
    "metCaterpillar": true,
    "numberOfTweedles": 2
};
OR 
var jsonString = 
    '{
        "firstName": "Alice",
        "lastName": "Wonderland",
        "metCaterpillar": true,
        "numberOfTweedles": 2
}';

Converting from JSON String to an Object

Code
var obj = JSON.parse(jsonString);

Converting from an Object to a JSON String

Code
var str = JSON.stringify(obj);
